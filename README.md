# 🚀 Cadastro de Usuários - Next.js + MongoDB

Sistema full stack de cadastro de usuários desenvolvido com Next.js, utilizando o App Router para integrar frontend e backend no mesmo projeto.

## 📌 Sobre o projeto

A aplicação permite o cadastro de usuários através de um formulário, enviando os dados para uma API interna que realiza validações e armazena as informações em um banco de dados MongoDB Atlas.

Este projeto foi desenvolvido com foco em aprendizado de integração entre frontend, backend e banco de dados, seguindo boas práticas de organização e uso de variáveis de ambiente.

---

## 🛠️ Tecnologias utilizadas

- Next.js
- TypeScript
- MongoDB Atlas
- Node.js
- CSS

---

## ⚙️ Funcionalidades

- Cadastro de usuários
- Validação de campos obrigatórios
- Verificação de e-mail já cadastrado
- Integração com banco de dados
- API REST com método POST

## 🔧 Como rodar o projeto

```bash
